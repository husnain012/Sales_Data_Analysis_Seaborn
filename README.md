# Sales Data Analysis

A business analytics project analyzing e-commerce sales data using Python, Pandas, and Seaborn.

## Analysis
- Sales performance by product category
- Monthly sales trends
- Top 5 most profitable orders
- Customer demographics (age distribution)
- Regional sales performance
- Sales-Profit-Rating correlation analysis

## Visualizations
1. **Bar Chart**: Sales by product category
2. **Line Plot**: Monthly sales trend
3. **Horizontal Bar**: Top 5 profitable orders
4. **Histogram**: Customer age distribution
5. **Bar Chart**: Regional sales performance
6. **Heatmap**: Correlation analysis

## Technologies
- Python 3
- Pandas - Data manipulation
- Seaborn - Data visualization
- Matplotlib - Plot customization
- NumPy - Numerical operations

## Usage
```bash
# Install dependencies
pip install pandas matplotlib seaborn numpy

# Run analysis
sales_data_analysis.py
